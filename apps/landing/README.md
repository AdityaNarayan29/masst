# Masst UI Landing Page

Marketing and landing page for the Masst UI component library.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 3.4
- **Animations**: GSAP
- **Icons**: Lucide React

## Development

```bash
# From monorepo root
pnpm dev:landing

# Or from this directory
pnpm dev
```

The app runs on [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
```

## Features

- Responsive design
- Dark mode support
- GSAP-powered animations
- Premium loading screen
- SEO optimized

## Deployment

This app is configured for deployment on Vercel. See [vercel.json](../../vercel.json) for configuration.
